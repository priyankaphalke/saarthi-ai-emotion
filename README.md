## 📖 Project Overview

**Saarthi AI** is an intelligent guidance platform designed to provide users with **AI-powered answers and emotional support**.

People often face confusion regarding **career decisions, studies, mental stress, and everyday problems**. Saarthi AI acts like a **digital mentor**, helping users by providing helpful suggestions and guidance.

The platform aims to make **AI-based guidance accessible, supportive, and easy to use**.

## 🧠 How It Works

1. User visits the website.
2. User asks a question or shares their concern.
3. The system analyzes the input.
4. AI generates a helpful response.
5. The user receives guidance instantly.
